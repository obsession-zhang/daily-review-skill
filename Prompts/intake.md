# 用户信息录入模板

## 基础信息（可选填，填了复盘更精准）

### 个人画像
- **职业/岗位**：__________
- **工作年限**：__________
- **团队角色**：__________（如：技术负责人、产品经理、独立贡献者等）
- **管理幅度**：__________（带几个人 / 不带人）
- **工作模式**：__________（远程 / 混合 / 坐班）

### 当前阶段
- **近期核心目标**：__________（如：推进某项目上线、带新人、转型等）
- **最大压力源**：__________
- **自我评价**：__________（你认为自己是哪类工作者？如：完美主义者、拖延型、行动派等）

### 复盘偏好
- **评判严格度**：__________（温和提醒 / 正常 / 严厉鞭策）
- **关注重点**：__________（效率 / 沟通 / 决策 / 情绪 / 全部）
- **输出语言**：__________（中文 / 英文）

## 当日素材提交

请按以下格式粘贴你的当日素材，可以混合多种类型：

```
【类型：聊天记录】
（粘贴聊天内容）

【类型：工作文档】
（粘贴文档内容或摘要）

【类型：会议录音转写】
（粘贴转写文本）

【类型：TODO清单】
（粘贴今日计划及完成情况）

【类型：邮件】
（粘贴邮件往来）

【类型：随手记/备忘录】
（粘贴任意文字记录）

【类型：其他】
（任何你觉得相关的素材）
```

## 补充信息（可选）

- **今天的心情/能量水平**：1-10 分
- **今天的意外事件**：__________
- **你想重点复盘的事情**：__________
- **你不想被评判的事情**：__________（复盘时会尊重此边界）
