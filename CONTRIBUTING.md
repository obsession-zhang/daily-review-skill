# Contributing to Daily Review Skill

感谢你的贡献！以下是参与方式。

## 如何贡献

### 1. 提交 Issue

- 发现复盘逻辑有偏差？开 Issue 描述具体场景
- 有新的输入类型想支持？开 Issue 讨论
- 想让 Mirror 更懂你的行业？分享你的复盘案例

### 2. 提交 PR

1. Fork 本仓库
2. 创建你的分支：`git checkout -b feature/xxx`
3. 提交更改：`git commit -m 'add: xxx'`
4. 推送分支：`git push origin feature/xxx`
5. 提交 Pull Request

### 3. 分享你的复盘案例

在 Discussion 区分享你的复盘报告（脱敏后），帮助改进 Skill 的准确性。

## 代码规范

- Prompt 文件使用 Markdown 格式
- Python 工具遵循 PEP 8
- 所有新增功能需附带示例

## 行为准则

- 尊重不同工作风格和价值观
- 复盘是自我观察，不是自我批评
- 保护隐私，不分享他人敏感信息
